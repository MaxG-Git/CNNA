# Diffenrent Commands used in CNNA


## Initialize Router and Reload
Start by Pressing `enter` on the keybaord. Afterwards Enter Privilaged EXEC Mode by using enable command:
```console
enable
```
or
```
en
```
<hr/>

Next Erase Prevoius Configurations
```
erase starup-config
reload
```
or
```
erase st
reload
```
